# Tip-Calculator-
Tip calculator using javascript

Tip generates according to your feedback 
1. 'Good' genrates 15% of tip
2. 'It is OK' genrates 10% of tip
3. 'Bad' genrates 5% of tip


Test Case 1:
Amount:1000
Feedback:It was OK
Spilt into:4
Result:Tip ₹25.00 each
       Toaltip: ₹100.00 

Test Case 2:
Amount:2000
Feedback:Bad
Spilt into:2
Result:₹50.00 each
